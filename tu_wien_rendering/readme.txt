Instructions:
1. Install MinGW (32 bit) from the given .exe file or from here: http://sourceforge.net/projects/mingw/files/latest/download?source=dlp
2. Tick the C++ box during the installation.
3. If you have installed it under a different directory than C:\MinGW, then modify the first line of c.bat!
4. Install IrfanView from the given.exe file or from here: http://portableapps.com/bouncer?t=http%3A%2F%2Fdownload.portableapps.com%2Fportableapps%2Firfanviewportable%2FIrfanViewPortable_4.35_Rev_2.paf.exe
5. Run c.bat.
6. Open the resulted image "image.ppm" in IrfanView.
+1. If you're under Linux, make sure to compile a 32 bit binary, i.e. use the -m32 flag.
